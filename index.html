Explore Collatz-style rules directly in the browser. Adjust a and b, then watch the graph that links each value to the result of y = ax + b for odd numbers and y = x/2 for even numbers.

How it works

  • Even numbers map to x/2; odd numbers map to ax + b.
  • Nodes attract their target and repel neighbors to keep the layout readable.
  • Colors depend on the highest value seen so far; higher values drift toward red.
  • If the rule grows past the selected upper bound, the simulation pauses.
